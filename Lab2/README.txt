Опис проекту
Додаток демонструє роботу з Java Stream API (JDK 24) для обробки масиву даних.
Реалізовано:
1. Генерацію нескінченного потоку об'єктів.
2. Власний Gatherer для фільтрації перших N входжень.
3. Групування та фільтрацію даних.
4. Власний Collector для збору статистики (мін, макс, середнє, відхилення).
5. Статистичний аналіз викидів (Outliers) за методом IQR.

Збірка та Запуск
Вимоги: JDK 22+ (з підтримкою Preview features).

Компіляція:
javac --enable-preview --release 24 Main.java

Запуск:
java --enable-preview Main
